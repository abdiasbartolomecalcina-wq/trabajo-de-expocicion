💳 NOTAS DE DÉBITO Y CRÉDITO

parte cencilla

📘 Introducción

Son documentos que se utilizan para corregir una factura sin modificarla directamente.

Se aplican cuando el monto necesita ajustarse, ya sea aumentando o disminuyendo.

benji

🔵 Nota de Crédito

Se usa cuando el monto debe disminuir.

  • Devolución de productos
  • Descuentos
  • Errores de cobro

Ejemplo: Compras por 200 Bs y devuelves un producto → te descuentan ese dinero.

Ronald

🔴 Nota de Débito

Se usa cuando el monto debe aumentar.

  • Cargos faltantes
  • Errores de precio
  • Intereses o recargos

Ejemplo: Compras por 100 Bs pero faltó el envío → se agrega un costo extra.

🧠 Idea clave

Crédito = baja el monto ⬇️

Débito = sube el monto ⬆️
